Küçük yaşlardan itibaren teknolojiyle iç içe büyüyerek SaaS uygulamaları, özgün web platformları, yapay zeka entegrasyonları ve sürükleyici 3D oyunlar üretiyorum. Kendi projelerimi A'dan Z'ye tasarlayıp yayınlayarak gerçek kullanıcı deneyimleri yaratmayı hedefliyorum.

*   ⚽ **Tutku Alanım:** Futbol analizleri, taktikler ve disiplin.
*   💻 **Geliştirme Alanlarım:** Modern SaaS platformları, web scraping, bilgisayarlı görü (computer vision), asenkron veritabanları ve 3D oyun geliştirme.
*   🧠 **Hedefim:** Küresel çapta ses getiren yapay zeka destekli SaaS ürünleri ve yenilikçi oyunlar sunmak.

### 📂 Öne Çıkan Projelerim

*   **Açıklama:** İçerik üreticileri için tasarlanmış, bilgisayarlı görü (thumbnail duygu analizi), ses/tempo analizi ve Groq/Gemini destekli AI Koç barındıran asenkron, tam teşekküllü bir SaaS analiz platformu.
*   **Teknolojiler:** Python, FastAPI, SQLite (aiosqlite WAL), OpenCV, Librosa, Gemini 2.0 Flash Vision, Llama-3.3-70B.

*   **Açıklama:** En büyük tutkum olan futbolu disiplinli analiz ve takip mekanizmalarıyla birleştiren özel bir web platformu.

#### 📈 RankVibe

*   **Teknolojiler:** Javascript, CSS3, Güvenli API Entegrasyonları.

#### 🎮 3D Strategy Game & BabaClutch
